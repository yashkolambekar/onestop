In this competitive world, sometimes there will be circumstances where you will need to show your creations, but at the same time you don't want to show your source code to anyone

This is by default in PHP, but to do this in HTML gets a bit tricky and uneasy
Here, I will be showing you how to hide your HTML and Javascript code


Step 1
The code given below will Restrict users from inspecting the elements by disabling the context menu (right click) and also disabling some keyboard actions such as F12, Ctrl + U, Ctrl + Shift + I and so on
It's okay to an extent to disbale these keys but disabling the context menu is not recommended 

// To be pasted in JS file
document.onkeydown = function(e) {
    if(event.keyCode == 123) {
        return false;
    }
    if(e.ctrlKey && e.shiftKey && e.keyCode == 'I'.charCodeAt(0)){
        return false;
    }
    if(e.ctrlKey && e.shiftKey && e.keyCode == 'J'.charCodeAt(0)){
        return false;
    }
    if(e.ctrlKey && e.keyCode == 'U'.charCodeAt(0)){
        return false;
    }
}


Now we have prevented the user from opening the inspection panel and from viewing the source code but if the user disables the Javascript, our code will not be executed and user can view the source code to prevent this we have to do something which will stop the user from disabling the Javascript

So we will make some changes to the code, that the webpage will be shown only if the Javascript is turned on
To do this we will give a class to elements with the same display property, for example the display property of header, section-one and footer is flex, we will give it a common class named shown-on-js

<!-- HTML Code-->
<header class="show-on-js"></header>
<div id="section-one" class="show-on-js"></div>
<footer class="show-on-js"></footer>

Now we will set the display property of this class to none, so the elements will not be shown initially 
/* CSS */
.show-on-js{
    display: none;
}

So now these elements will not be shown on the webpage initially, now we will make it visible through Javascript, so now it will be shown only if the Javascript is turned on 
// Javascript code
document.querySelectorAll(".show-on-js").display = block;

Make sure that all the elements are suitable for the particular display property

This is indeed not a full-proof solution to the problem, but this makes the process so complex and layered that most of the times, the person trying to get a sneak of your code will give up and your site will be safeguarded to an extent.

Having spinners or buttons on a number input makes it look odd for a webpage and also ruins the text align in that particular input field, to avoid this copy paste the code given below and make changes in the selector as per your case 

/* CSS */
input[type=number]::-webkit-inner-spin-button, 
input[type=number]::-webkit-outer-spin-button { 
  -webkit-appearance: none; 
  margin: 0; 
}
