Everything about Colors

This page will be a solid guide for you to deal with colors and some tools to make your life easier


To start with let's learn first about the basic of Colors
Red, Green and Blue are primary colors every and other color that we see is made by mixing these three colors in sepcific quantities

In HTML we can specify colors with by specifying their names, by specifying their hex values or by specifying the rgb value, out of all these rgb values method is best for us to learn about colors

It looks like this,
rgb(255, 0, 0);, rgb(0, 255, 0);, rgb(0, 0, 255);

To get an idea of how it works, rgb(rrr, ggg, bbb) here the r's inside the parenthesis or the brackets define the value of red in that particular color, it cam vary from 0 to 255 it's quite self explainatory that 0 means no red will be included and 255 means the most it can
The same goes with the g's for green and b's for blue
If we turn everything to 0, the color will be black and if we turn everything to 255, the color will be white
